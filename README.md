# بِسْمِ اللَّهِ الرَّحْمَنِ الرَّحِيمِ
# Mon Deen au Quotidien

Application islamique complète — 100% locale, vie privée totale.

## 🚀 Lancement rapide

### Windows (PowerShell)
1. Clique droit sur `Lancer.ps1`
2. Sélectionne "Exécuter avec PowerShell"
3. L'app s'ouvre dans ton navigateur

### Si PowerShell bloque :
```
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned
```

### Manuel
```bash
npm install
npm run dev
```

## 📱 Sur Android
1. Connecte ton téléphone au même Wi-Fi
2. Dans le terminal, note l'adresse IP affichée (ex: `192.168.1.x:3000`)
3. Ouvre cette adresse dans Chrome Android
4. Menu → "Ajouter à l'écran d'accueil" → installée comme APK !
